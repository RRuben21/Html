Mobile Web Design Best Practices

Design Considerations

Mobile web users are typically on-the-go, need information quickly and they are easily distracted. A web page that is optizmized for mobile access should try to serve these needs.

  • Optimize Layout for Mobile Use
  • Optimize Navigation to Mobile Use
  • Optimize Graphics for Mobile Use
  • Optimize Text for Mobile Use

    • CSS3 Media Queries

    There are two approaches on how to select the CSS that should be applied to the device.

  • Media Query Example Using a Link Element
  • <link href="nameOfStyleSheet.css" rel="stylesheet" media="only screen and (max-width: 480px)">
  • Media Query Example Using an @media Rule
  • media only screen and (max-width: 480px){ /* css rules; */ }
